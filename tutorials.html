How to Use Our Map

Here you will find a list of all features and tools available with our app, as well as tutorials on how to use them.

Navigation

This map comes equipped with standard map navigation tools, such as zooming, panning, and reset default view. A "My Location" feature is also available for use. These can be found in the top left corner of the map.

Search

The search bar, which can also be found in the top left corner of the map next to the navigation tools, allows for address and place name search. Once an address or place is entered, the map will automatically zoom into the selected location.

Legend

The map legend can be found as the first widget in the top right corner of the app. Click to toggle it on and off.

Layer List

The layer list can be found as the second widget in the top right corner of the app. Click to open the list and select layer visibility.

Filters

The map comes with two filters for the data: Types of Historic Places and Area of Significance. These can be found as the first two widgets in the bottom left corner of the map. Click the filters and choose which data points to appear on the map.

Near Me

This tool searches for all properties listed on the National Registry of Historic Places within a certain distance. Start by selecting a location with the search bar, by clicking the Current Location button, or by selecting a custom point with the Set Location button. Then, choose a distance from 0 to 200 miles using the provided slider. All results within the specified distance will appear in the tool window.

Results may not appear if the data layer is not visible in the Layer List pane.
