How to Use Our Map

Here you will find a list of all features and tools available with our app, as well as instructions on how to use them.

Navigation

This map comes equipped with standard map navigation tools, such as zooming, panning, and reset default view. A "My Location" feature is also available for use. These can be found in the top left corner of the map.

Search

The search bar, which can also be found in the top left corner of the map next to the navigation tools, allows for address and place name search. Once an address or place is entered, the map will automatically zoom into the selected location.

Legend

The map legend can be found as the first widget in the top right corner of the app. Click to toggle it on and off.

Layer List

The layer list can be found as the second widget in the top right corner of the app. Click to open the list and select layer visibility.

Filters

The map comes with two filters for the data: Types of Historic Places and Area of Significance. These can be found as the first two widgets in the bottom left corner of the map. Click the filters and choose which data points to appear on the map.

Near Me

This tool searches for all properties listed on the National Registry of Historic Places within a certain distance and can be found as the third widget in the bottom left corner of the map. Start by selecting a location with the search bar, by clicking the Current Location button, or by selecting a custom point with the Set Location button. Then, choose a distance from 0 to 200 miles using the provided slider. All results within the specified distance will appear in the tool window.

Results may not appear if the data layer is not visible in the Layer List pane.

Create Buffers

This tool creates circles around selected points on the map and can be found as the fourth widget in the bottom left corner of the map. Start by defining the circle's size by selecting Radius or Diameter, then specifying a distance (and its units). Lastly, easily select the center point using the Set Location button and clicking on the map. The circle tab also provides a Distance Calculator function. To use this, start by entering a time and rate, as well as the desired associated units. The radius or diameter will auto-populate with the calculated distance.

The tool also allows for the customization of the symbol and label settings. Click the tab at the bottom of the window to begin changing the appearance of the circles and labels.

Click the Clear Graphics button to remove any previous buffers.

Get Directions

This tool generates a route from selected start, middle, and end points and can be found as the fifth widget in the bottom left corner of the map. Start by selecting a beginning location. To select a point on the map, toggle the Add Stops button below the last stop. Then, simply click in the search bar and click anywhere on the map. The address will auto-populate the search bar as a stop on the route. To add more waypoints, click the ADD function below the last stop. Lastly, select the method of travel from the provided options and click the GET DIRECTIONS button.

The tool also allows for the specification of the departure time and if the journey requires a return to the start.

Use of this tool requires access to an ArcGIS Online account.
